

## Giới thiệu
FarmLife - Một tựa game 2d đơn giản được viết bằng C++ và SFML.

## Cách cài đặt

### Lưu ý:
- Cần cài đặt Visual Studio 2022 trở lên.
- Sau khi clone repo Visual Studio sẽ tự build từ file CMake.

1. Clone repo
   ```sh
   git clone https://github.com/DaLziel9901/Game_FarmLife
   ```
2. Mở project bằng Visual Studio 2022.
3. Ở góc trên bên phải Debug Target, chọn mục farm-life.exe
4. Nhấn F5 / Start để chạy chương trình.
